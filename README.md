# Appointment Booking Web App

This project is an appointment booking web application for a service-based business.

Project Goal
Build a web application that supports:
1. Service provider journey
    * Configure services
    * Configure staff members
    * Configure staff schedules
    * View booked appointments
2. Validate
    * Select a service
    * Choose a date
    * View available time slots
    * Book an appointment

Test
The app will preload sample data including:
* Services
* Staff members
* Staff schedules
* Existing booked appointments
